# Raw Documents

This directory is for storing PDF files and other raw documents
that need to be extracted to Markdown format.

## Purpose
- Store rulebooks, card lists, and other reference documents
- Extract content to Markdown for integration with the project docs
- Keep original files for reference and re-extraction if needed

## Current Status
- No files yet — add PDFs here when available

## Extraction Workflow
1. Place PDF in this directory
2. Extract content (manual or tool-assisted)
3. Save as Markdown in appropriate `docs/` subdirectory
4. Update references in `docs/index.md`
