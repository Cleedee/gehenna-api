# Checklist de Desenvolvimento - Trends

## 1. Análise Temporal
- [ ] Evolução do meta ao longo do tempo
- [ ] Cards que subiram/desceram em popularidade
- [ ] Clãs dominantes por período

## 2. Análise por Formato
- [ ] Comparar 2R+F vs 3R+F
- [ ] Quais cards funcionam melhor em cada formato

## 3. Análise de Arco (Arquétipos)
- [ ] Identificar arquétipos (bleed, vote, combat, toolbox)
- [ ] Combinações de cards característicos

## 4. Recomendações
- [ ] Sugerir cards para montar deck
- [ ] Identificar gaps no acervo local

## 5. Web UI
- [ ] Exibir tendências na interface
- [ ] Rankings visuais
- [ ] Gráficos de evolução
